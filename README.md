# RFM Customer Segmentation Analysis

## 📊 Project Overview
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis on an e-commerce dataset.

## 🛠 Tools Used
- SQL (SQLite)
- Power BI

## 📈 Key Insights
- High-value customers contribute majority of revenue
- At-risk customers show potential churn
- Loyal customers provide stable revenue
- Revenue distribution is skewed (Pareto principle)

## 📊 Dashboard Features
- Customer segmentation (RFM)
- Revenue analysis by segment
- Recency vs Monetary scatter plot
- Top 10 high-value customers

## 📁 Files Included
- Data (CSV)
- SQL queries
- Power BI dashboard

## 🎯 Business Impact
Helps businesses identify high-value customers and design targeted marketing strategies.
